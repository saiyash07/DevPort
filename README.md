# DevPort 🔌

A lightweight native macOS menu bar utility that monitors active listening TCP ports on localhost, showing you what services are running, opening them in the browser, or killing them with a single click.

## Features
- **Dynamic Menu Bar Text**: Shows the exact number of active local servers (e.g. `🔌 3`).
- **Interactive Actions**: Click on any port to open it in your browser or kill its process.
- **Lightweight & Native**: Written in Swift using AppKit (zero external dependencies).

## Installation & Build
```bash
chmod +x build.sh
./build.sh
./DevPort &
```
